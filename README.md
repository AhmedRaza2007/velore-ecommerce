## ✨ Features

- **🛍️ Complete E‑Commerce Experience** - Home, Shop, Categories, Collections, Product Details, Cart, Checkout, Wishlist
- **👥 User Authentication** - Login & Register pages
- **🔍 Advanced Filtering & Sorting** - Filter by category, brand, price, size, color, rating; Sort by featured, newest, best‑selling, price, rating
- **⚡ Smooth Interactions** - GSAP animations, Lenis smooth scrolling
- **📱 Fully Responsive** - Perfect on mobile, tablet, and desktop
- **🎨 Premium Design** - Inspired by Apple, MR PORTER, COS, SSENSE
- **❤️ Wishlist & Cart** - Save favorites and manage your shopping bag
- **📧 Newsletter Signup** - Stay updated with new arrivals
- **💬 Info Pages** - About, Contact, FAQ, Privacy Policy, Terms & Conditions

## 🌐 Deployment Options

### Option 1: GitHub Pages
1.  Push your code to a GitHub repository
2.  Go to **Settings > Pages** in your repo
3.  Under **Source**, select "Deploy from a branch"
4.  Choose your main branch and click **Save**
5.  Wait a few minutes for your site to go live!

### Option 2: Vercel (Recommended)
1.  Sign up / log in to [Vercel](https://vercel.com)
2.  Click "New Project" and import your GitHub repo
3.  Click "Deploy" - done!

### Option 3: Netlify
1.  Sign up / log in to [Netlify](https://netlify.com)
2.  Connect your GitHub repo
3.  Click "Deploy site" - that's it!
