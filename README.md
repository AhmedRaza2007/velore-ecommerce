# VELORÉ - Luxury E‑Commerce Website

> Experience the world of luxury fashion and lifestyle. VELORÉ brings you the finest curated pieces from around the globe.

---

## ✨ Features

- **🛍️ Complete E‑Commerce Experience** - Home, Shop, Categories, Collections, Product Details, Cart, Checkout, Wishlist
- **👥 User Authentication** - Login & Register pages
- **🔍 Advanced Filtering & Sorting** - Filter by category, brand, price, size, color, rating; Sort by featured, newest, best‑selling, price, rating
- **⚡ Smooth Interactions** - GSAP animations, Lenis smooth scrolling
- **📱 Fully Responsive** - Perfect on mobile, tablet, and desktop
- **🎨 Premium Design** - Inspired by Apple, MR PORTER, COS, SSENSE
- **❤️ Wishlist & Cart** - Save favorites and manage your shopping bag
- **📧 Newsletter Signup** - Stay updated with new arrivals
- **💬 Info Pages** - About, Contact, FAQ, Privacy Policy, Terms & Conditions

---

## 🚀 Quick Start

1.  **Clone the repository**
    ```bash
    git clone https://github.com/AhmedRaza2007/velore-ecommerce
    cd velore-ecommerce
    ```
2.  **Open in browser** - Just open `index.html` in your browser!

## 🌐 Deployment

This project is set up for **GitHub Pages** auto-deployment! Just:
1.  Go to your repo: https://github.com/AhmedRaza2007/velore-ecommerce
2.  Click **Settings > Pages**
3.  Under **Build and deployment > Source**, select **GitHub Actions**
4.  That's it! Your site will deploy automatically on every push!

Your live URL will be: https://ahmedraza2007.github.io/velore-ecommerce
